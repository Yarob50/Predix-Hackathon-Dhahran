 **:star: :one: Prototype built for Predix Hackathon - Dhahran - 2017 which was chosen to be the first place winner!:trophy::crown:**
 
The theme of the hackathon was about healthcare and we were asked to build a system that collects information gathered from different predix machines to be displayed in a dashboard that shows all the needed information to the concerned people in the Ministery of Health or in the health sector in general.
